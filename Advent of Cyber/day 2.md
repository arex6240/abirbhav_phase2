# Phishing

This task introduces the concept of phishing attacks. involves creating fake login page. this fake login page is hosted on `server.py` which listens on the port 8000. The Social-Engineer Toolkit (SET) helps automate crafting and sending such phishing emails by allowing attackers to configure sender details, SMTP routing, and message content thru the linux terminal.
